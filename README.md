# Smart Restaurant Order Management System

A Flask-based restaurant order management system with QR-enabled customer ordering, admin management, and kitchen workflow.

## Features

- Customer-facing ordering via table QR code
- Menu browsing, cart management, checkout, and order status
- Admin dashboard for:
  - menu item management
  - table management
  - user management
  - order tracking
  - sales / status reports
- Kitchen dashboard for order preparation workflow
- Automatic seed data for admin/kitchen users, menu items, and tables

## Default access
